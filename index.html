禁毒开源 · 都昌禁毒云展馆

一个可静态运行的禁毒教育小程序项目展示页。你可以直接替换占位图，快速对外展示项目功能与落地效果。

查看 README 文档 查看静态运行说明

项目模块

云展馆 互动体验 知识答题 VR导览 积分礼品

功能展示

云展馆导览

8 个主题展区，支持展区互动、签名承诺、学习行为积分。

互动体验

故事线 + 游戏体验，覆盖青少年、家庭、职场等教育场景。

知识答题

内置题库与答题模式，统计正确率并与积分系统联动。

VR + 语音导览

支持本地静态数据运行，方便离线演示与功能验收。

快速开始

  1. 微信开发者工具导入目录：miniprogram
  2. 确认 miniprogram/config/app.js 中 mockData: true
  3. 直接编译运行（无需后端）
